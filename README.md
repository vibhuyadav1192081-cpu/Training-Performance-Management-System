# 🎓 Training Performance Management System

A Professional Training Performance Management System built using Python and Streamlit.

## Features

- Student Performance Dashboard
- Quiz Analytics
- AI Performance Insights
- Automatic Ranking
- Grade Card PDF Generation
- QR Code Verification
- Email Automation
- Interactive Dashboard

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- ReportLab
- NumPy
